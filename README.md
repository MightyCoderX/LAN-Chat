# LAN Chat
A simple socket chat made with [socket.io](https://socket.io/)

## Features
- [x] Send and receive messages in a group chat
- [x] Embed images
- [x] User typing info message
- [x] Join/Leave messages
- [x] Message Timestamp
- [x] Image preview
- [ ] Highlight links
- [ ] Attach documents
- [ ] User Mentions
- [ ] Send voice messages
- [ ] Embed videos
- [ ] Notifications
- [ ] Private rooms
